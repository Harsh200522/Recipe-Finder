## 🔐 Deployment Instructions

### 1. Install & Test Locally
```bash
npm install          # Install dependencies
npm run dev          # Test locally at http://localhost:5173
npm run build        # Build for production
npm run preview      # Preview production build
```

### 2. Deploy to Production
Choose one:
- **Vercel** (recommended): `npm i -g vercel && vercel`
- **Netlify**: `npm i -g netlify-cli && netlify deploy`
- **Firebase**: `npm i -g firebase-tools && firebase deploy`
- **Any hosting**: Upload `dist/` folder via FTP/control panel

### 3. Configure Domain
- Point domain to hosting
- Enable HTTPS (automatic with most hosts)
- Test in browser

### 4. Submit to Google
```
Google Search Console:
1. Add domain
2. Verify ownership
3. Upload /sitemap.xml
4. Request indexing

Google AdSense:
1. Apply for account
2. Paste AdSense code (in index.html)
3. Wait for approval
```

## 🆘 Troubleshooting

### Q: Articles not showing?
A: Check `src/data/blog-articles.js` is properly formatted

### Q: Styles look broken?
A: Run `npm run build` and check all CSS files exist

### Q: Routing errors?
A: Verify all routes in `App.jsx` match your components

### Q: AdSense not working?
A: Replace `ca-pub-YOUR-CODE` in `index.html` with your real code

### Q: Page loads slow?
A: Compress images, reduce dependencies, enable caching
